# LatentMDM (LP-MDM)

Code for the paper's LatentMDM experiments on TinyGSM: training
(`train_lpmdm.py`), sampling (`sampling.py`), and evaluation
(`eval_lpmdm.py`, `eval/`), with configs under `yaml_files/`.

Environment: `environment.yml`. See the repository root README for the paper
and the companion `FlexMDM/` codebase.
